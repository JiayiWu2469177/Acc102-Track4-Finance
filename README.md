# ACC102 - Track 4: Interactive Financial Data Analysis Tool
Course Assignment | Streamlit + Python Data Product

## Project Overview
This project delivers an interactive data product designed for analyzing corporate financial performance. Built with Python and Streamlit, the tool enables users to efficiently explore, visualize, and interpret key financial metrics such as Return on Equity (ROE) and Net Profit Margin.

## Key Features
- Data Visualization: Auto-generates clear bar charts to compare ROE across companies.
- Interactive Interface: Easy-to-use sidebar for data selection and configuration.
- Financial Insights: Built-in explanations to help non-technical users understand the business meaning of key metrics.
- Data Compatibility: Supports standard CSV/Excel formats, compatible with data from CSMAR or WRDS databases.

## Technical Structure
The repository contains the following core files:
1. **app.py**: Main application script containing the Streamlit dashboard logic and visualization code.
2. **data.csv**: Sample dataset for demonstration, including historical financial performance data.
3. **requirements.txt**: List of Python dependencies required for deployment.
4. **README.md**: Project documentation (this file).
